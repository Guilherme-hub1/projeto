# Sistema de Recomendação em Python

Projeto desenvolvido em Python utilizando similaridade de cosseno para recomendar itens com base nas preferências do usuário.

## Sobre o Projeto

Este projeto foi criado com o objetivo de demonstrar o funcionamento de um sistema de recomendação simples utilizando:

* Python
* NumPy
* Scikit-learn
* IPyWidgets
* Jupyter Notebook

O sistema compara vetores de dados e calcula a similaridade entre eles para gerar recomendações personalizadas.

## Tecnologias Utilizadas

* Python 3
* NumPy
* Scikit-learn
* IPyWidgets
* Jupyter Notebook

## Funcionamento

O projeto utiliza:

* Vetores numéricos
* Similaridade de cosseno
* Interface interativa com widgets

A partir das escolhas do usuário, o sistema calcula quais itens possuem maior semelhança e retorna recomendações.
